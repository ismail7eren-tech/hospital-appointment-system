🏥 Hastane Yönetim Sistemi

Modern web teknolojileri kullanılarak geliştirilmiş bu proje, hastane süreçlerini dijital ortamda yönetmeyi amaçlayan basit ama işlevsel bir randevu yönetim sistemidir. Kullanıcıların hasta, doktor ve randevu işlemlerini kolayca gerçekleştirebilmesini sağlar.

📌 Proje Hakkında

Bu uygulama, backend ve frontend entegrasyonunu öğrenmek ve gerçek dünya problemlerine çözüm üretmek amacıyla geliştirilmiştir. Sistem, kullanıcı dostu bir arayüz ile temel hastane operasyonlarını yönetmeye olanak tanır.

Projede:

Veri kalıcılığı için SQLite

Backend tarafında Flask

Arayüz tarafında HTML, CSS ve JavaScript

kullanılmıştır.

🚀 Özellikler
👤 Hasta Yönetimi

Yeni hasta ekleme (isim, soyisim, TC)

Verilerin veritabanında kalıcı olarak saklanması

🩺 Doktor Yönetimi

Doktor ekleme

Uzmanlık alanını hazır listeden seçme (manuel giriş yok)

Standart veri yapısı ile hatasız kayıt

📅 Randevu Sistemi

Hasta ve doktora göre randevu oluşturma

Uzmanlığa göre doktor filtreleme (dinamik seçim)

Randevu listeleme

Randevu silme (onaylı işlem)


🛠️ Kullanılan Teknolojiler
Teknoloji	Açıklama
Python	Backend geliştirme
Flask	Web framework
SQLite	Hafif veritabanı
HTML	Sayfa yapısı
CSS	Tasarım
JavaScript	Dinamik işlemler
